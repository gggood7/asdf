## 리눅스 명령어

1) ___top 명령어___

top 명령어는 실시간으로 현재 os의 상태를 나타내주는 CLI 어플리케이션

(CPU의 사용율 체크 / CPU, Memory, Process)

* top 명령어 옵션

`-b` : batch 모드

`-n` : 실행 주기 설정

`shift + p` : CPU 사용률 내림차순

`shift + m` : Memory 사용률 내림차순

`shift + t` : 프로세스가 돌아가고 있는 시간 순  등...

Page up/down을 통해 페이지 이동이 가능
